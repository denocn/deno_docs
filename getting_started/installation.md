## Updating
>>>>>>> 38a19024c4c227f7964daf86e43db32bc062ec25

如需更新旧版本的 Deno，你可以运行：

```shell
deno upgrade
```

此操作会从
[github.com/denoland/deno/releases](https://github.com/denoland/deno/releases)
下载最新并解压，并使用解压后的文件替换你当前的可执行文件。

你也可以使用此工具来安装特定版本的 Deno：

```shell
deno upgrade --version 1.0.1
```
